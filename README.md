# Scanoff

## Description

Scanoff performs a full cycle of web application reconnaissance and security testing:

- Subdomain enumeration
- Port scanning
- WAF identification
- Technology detection (CMS, frameworks)
- Swagger/OpenAPI parsing
- Vulnerability scanning (SQLi, XSS, LFI, RCE)
- Content analysis for sensitive data
- Screenshot capture of web pages
- Service fingerprinting via banners
- Dependency vulnerability checking
- Sending Telegram notifications
- Report export (PDF, HTML, JSON, CSV)
- Web dashboard for results visualisation
- Database storage of scan results
